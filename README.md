**Offline First Movies App**

This is an offline-first moviews app. Pulls moview information from [[themoviedb.org](https://www.themoviedb.org/)]

Concepts used:

1. MVVM UI architecture
2. Dependency Injection with Hilt
3. Retrofit
4. Room DB
5. Jetpack Compose
